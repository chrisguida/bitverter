# Bitverter Roadmap

Complete basic CLI functionality
*	Ability to convert from btc to mbtc, bits, sats and vice versa

Complete Rust web server
*	warp as a server
*	reqwest for APIs
*	serde for parsing responses
*	maud for HTML templating
